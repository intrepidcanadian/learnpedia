# learnpedia

A personal wikipedia for making great AI videos — prompts, camera angles, lighting, character consistency, motion, post-production, and more.

Built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Topics Covered

- **Prompting** — Prompt engineering, negative prompts, templates, scene description, style transfer, color grading
- **Camera & Composition** — Angles, movements, lens choices, composition rules, depth of field
- **Lighting** — Three-point lighting, natural light, mood lighting
- **Character Consistency** — Reference sheets, face consistency, wardrobe consistency
- **Motion & Animation** — Motion dynamics, frame rates
- **Post-Production** — Artifacts, workflow, upscaling, audio
- **Tools & Platforms** — AI video tool comparison, image-to-video
- **Workflows** — Multi-shot workflows, shot list planning
