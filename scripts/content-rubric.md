# Content Quality Rubric

Score each article 1-5 on each dimension. The rubric is designed for AI video production wiki articles.

## Dimensions

### 1. Factual Accuracy (weight: 3x)
- **5**: All claims are correct and current (platform capabilities, technical specs, pricing, availability)
- **4**: Minor imprecisions that don't mislead (e.g., slightly outdated version numbers)
- **3**: One notable inaccuracy or outdated claim that could mislead
- **2**: Multiple inaccuracies or significantly outdated information
- **1**: Fundamentally incorrect core claims

### 2. Completeness (weight: 2x)
- **5**: Covers all essential aspects of the topic; no major gaps
- **4**: Covers most essential aspects; 1 notable gap
- **3**: Covers the basics but missing 2-3 important subtopics
- **2**: Significant gaps in coverage; feels like a stub
- **1**: Barely introduces the topic

### 3. Prompt Examples (weight: 2x)
- **5**: Multiple copy-pasteable prompt examples with context on when/why to use each
- **4**: Good prompt examples covering main use cases
- **3**: Some prompt examples but missing key scenarios
- **2**: One or two basic examples only
- **1**: No prompt examples (or article type doesn't need them — score N/A)

### 4. AI-Specific Guidance (weight: 2x)
- **5**: Clearly distinguishes how AI video differs from traditional filmmaking; platform-specific tips; reliability notes
- **4**: Good AI-specific guidance with some platform notes
- **3**: Mentions AI context but reads mostly like generic filmmaking advice
- **2**: Minimal AI-specific guidance
- **1**: No AI-specific guidance; could be a traditional filmmaking article

### 5. Cross-References (weight: 1x)
- **5**: Links to all relevant related articles; no dead-end topics
- **4**: Links to most related articles
- **3**: Some cross-references but missing obvious connections
- **2**: Minimal cross-references
- **1**: No cross-references; isolated article

### 6. Actionability (weight: 1x)
- **5**: Reader can immediately apply the advice; clear steps, specific values, concrete guidance
- **4**: Mostly actionable with some vague areas
- **3**: Mix of actionable and theoretical content
- **2**: Mostly theoretical; reader needs to figure out application
- **1**: Purely theoretical; no practical guidance

## Scoring

**Weighted score** = (Accuracy * 3 + Completeness * 2 + Prompts * 2 + AI-Specific * 2 + Cross-Refs * 1 + Actionability * 1) / 11

**Rating thresholds:**
- 4.5+ = Excellent
- 3.5-4.4 = Good
- 2.5-3.4 = Needs Work
- < 2.5 = Poor
